README에 커버리지 예시가 있으나 `pyproject.toml`의 `dev`에는 `pytest-cov`가 없습니다. 설치 여부를 확인합니다.



이 저장소는 **pytest**만 `dev`에 넣어 두었고, **`pytest-cov`는 아직 설치되어 있지 않습니다.** 커버리지를 보려면 먼저 플러그인을 설치해야 합니다.

## 1. 설치

프로젝트 루트에서 (가상환경이 있으면 활성화한 뒤):

```powershell
pip install pytest-cov
```

또는 한 번에:

```powershell
pip install -e ".[dev]"
pip install pytest-cov
```

(원하면 `pyproject.toml`의 `dev`에 `pytest-cov`를 추가해 두면 `pip install -e ".[dev]"`만으로 충분합니다.)

## 2. 터미널에서 커버리지 실행

`pyproject.toml`에 선언된 패키지는 `boundary`, `entity`, `magicsquare`(및 `magicsquare.screen`)입니다. 예시는 다음과 같습니다.

**요약만 (터미널):**

```powershell
cd c:\DEV\Magic-Square_20
python -m pytest --cov=entity --cov=boundary --cov=magicsquare --cov-report=term-missing
```

- `--cov-report=term-missing`: 커버되지 않은 줄 번호를 같이 출력합니다.

**PRD/README에서 말하는 80% 게이트까지 한 번에:**

```powershell
python -m pytest --cov=entity --cov=boundary --cov=magicsquare --cov-fail-under=80 --cov-report=term-missing
```

README에 나온 `--cov=control`은 이 레포의 `setuptools` 패키지 목록에는 없으므로, 실제로 `control` 디렉터리가 생기면 그때 `--cov=control`을 추가하면 됩니다.

## 3. HTML 리포트 (브라우저에서 보기)

```powershell
python -m pytest --cov=entity --cov=boundary --cov=magicsquare --cov-report=html
```

실행 후 프로젝트 루트에 생기는 `htmlcov\index.html`을 브라우저로 열면 파일·줄 단위로 확인할 수 있습니다. (`.gitignore`에 `.coverage`가 있으므로 측정 DB는 커밋되지 않습니다.)

## 4. 특정 테스트만 돌리면서 커버리지

```powershell
python -m pytest tests\entity --cov=entity --cov-report=term-missing
```

정리하면, **`pytest-cov` 설치 → `python -m pytest`에 `--cov=...`와 `--cov-report=`** 조합으로 확인하면 됩니다.

## 사용 방법

1. **커버리지 전용 창** (표 + 로그 + 행 선택 시 미커버 줄 번호)

```text
pip install -e ".[gui]"
python -m magicsquare --coverage
```

2. **기존 pytest 창에서** **「커버리지 표…」** 를 누르면 같은 표 창이 하나 더 뜹니다.

```text
python -m magicsquare --tests
```

`[gui]`에 `pytest`, `pytest-cov`를 넣어 두었기 때문에 GUI만 설치해도 커버리지 실행이 됩니다.

## 동작

- 저장소 루트에서 `pytest tests`를 `entity` / `boundary` / `magicsquare`에 대해 돌리고, coverage **JSON**을 임시 파일로 받은 뒤 삭제합니다.
- 위쪽 **표**: 파일 경로, 구문 수, 실행(커버된 줄 수), 누락, 커버 % (정렬 가능).
- **합계**는 표 아래 라벨에만 표시해 정렬과 섞이지 않게 했습니다.
- 아래 **텍스트**: pytest 출력; 표에서 파일 행을 고르면 그 파일의 **미커버 줄 번호**를 이어서 보여 줍니다.
